# 🚀 vue-shop · 后台管理前端（Vue3 + Vite + Vue Router + Axios）

这是一个基于 **Vue3 + Vite** 构建的前端后台管理系统，作为旧 JavaWeb（Servlet + JSP） 项目的现代化重构版本。  
整个项目采用 **前后端分离架构**：  
- 后端使用 Spring Boot 提供 REST API  
- 前端使用 Vue3 渲染页面、请求接口

本项目目前实现了后台首页布局 & 用户管理模块，并持续扩展中。

---

## 🛠 技术栈

| 技术 | 用途 |
|------|------|
| **Vue 3** | 构建前端页面（Composition API + SFC） |
| **Vite** | 项目构建工具，极速热更新 |
| **Element Plus** | 后台管理常用 UI 组件库 |
| **Axios** | 调用后端 Spring Boot REST API |

本项目已集成 Element Plus，后续后台管理页面（如用户管理、分页、表单、弹窗等）将逐步替换为基于 Element Plus 的现代化组件，以提高整体 UI 体验与开发效率。
